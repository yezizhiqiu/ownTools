## jsDataType.js ##
+ isFunction 是否为函数
+ isWindow 检测是否为window对象
+ isPlainObject 是否为纯粹的对象{}（数组和正则等都不是纯粹的对象）
+ isEmptyObject 是否为空对象
+ isArrayLike 是否为数组或者类数组
